# NairaTrack 💰

> Smart personal finance tracker built for Nigerian students.

Built for the **IWD26 Lovelace Challenge** — deployed on CreateOS.

## What it does

NairaTrack helps Nigerian students track daily expenses, understand spending patterns, and make smarter money decisions — all in one clean dashboard.

## Features

- 📊 **Spending Dashboard** — donut chart by category, 7-day line chart
- 💡 **AI-Powered Insights** — personalized tips based on your actual spending
- ⚡ **Quick Log** — tap a category, enter amount, done in 5 seconds
- 📈 **Budget Progress** — see exactly how much you have left this month
- 🧾 **Transaction History** — full log of every expense
- 🇳🇬 **Built for Nigeria** — ₦ Naira, Nigerian context, Nigerian student life

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Recharts
- Deployed on CreateOS
